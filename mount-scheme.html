Mount Scheme

Mount scheme `mount://` did the work which mount a device and then change directory to the mount point.

Scheme Struct

Mount scheme simply contains a device scheme url in the url fragment:

mount://#<url to be mounted>

the url which is going to be mounted should be a `udisks` scheme url, or a `device` scheme url.

Example Urls

mount://test#udisks:///org/freedesktop/UDisks2/block_devices/sda1

This will mount /dev/sda1 and then open it.
