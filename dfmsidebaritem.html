Detailed Description

DFMSideBarItem is the item inside DFMSideBar

DFMSideBarItem is the base class for the sidebar items. All type of items inside the sidebar is derived from DFMSideBarItem. DFMSideBarItem provide the interfaces to manage the item state.

See also DFMSideBar.

[slot] void DFMSideBarItem::setAutoOpenUrlOnClick(bool autoCd)

Disable auto change directory (`cd`) on sidebar item is clicked autoCd

The `DFMSideBarItem`'s default click behavior is change directory to `url()` path. If you are going to write a custom item and don't want the default behavior, then set it to false and handle `clicked()` signal by yourself.

Note: Setter function for property autoOpenUrlOnClick.

See also autoOpenUrlOnClick(), url(), and clicked().
