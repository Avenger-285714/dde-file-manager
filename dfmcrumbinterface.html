Detailed Description

DFMCrumbInterface is the interface for crumb item management of DFMCrumbBar

DFMCrumbInterface is the interface for crumb item management for DFMCrumbBar, and should be implemented as a controller. Each different URL scheme got a corresponding controller for creating crumb items. DFMCrumbManager holds all avaliable crumb controllers.

DFMCrumbInterface provides all neccessary interfaces for crumb management and some URL processing utils functions.

See also DFMCrumbBar and DFMCrumbManager.

[virtual] void DFMCrumbInterface::requestCompletionList(const DUrl &url)

Start request a completion list for address bar auto-completion.

url The base url need to be completed.

Since completion list can be long, so we need do async completion. Calling this function will start a completion request and the completion list item will be sent via signal completionFound. When user no longer need current completion list and the transmission isn't completed, you should call cancelCompletionListTransmission. When transmission completed, it will send completionListTransmissionCompleted signal.

See also completionFound, completionListTransmissionCompleted, and cancelCompletionListTransmission.
